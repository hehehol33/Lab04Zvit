Десктопна версія (ширина більше 1024px)

У десктопній версії контент показується у великому форматі. Зображення мають великий розмір для повного відображення на широких екранах, шрифти таблиць та списків достатньо великі для зручного перегляду. Елементи списку та таблиці також мають більший відступ, щоб забезпечити комфортне візуальне сприйняття.

Планшетна версія (від 601px до 1024px)

Для планшетів контент стає більш компактним. Зображення займають більше місця по ширині, але мають збережені пропорції для забезпечення коректного відображення. Шрифти таблиць та списків зменшені для економії простору, а елементи списку та кнопки стають менш громіздкими.

Мобільна версія (ширина до 600px)

У мобільній версії контент повністю адаптований під вузькі екрани. Зображення масштабуються під ширину екрана, щоб не було горизонтальної прокрутки, таблиці та списки відображаються у більш компактному вигляді. Шрифти також зменшені для забезпечення читабельності, а кнопка надсилає форму, розтягуючись на всю ширину екрану для зручності взаємодії.
